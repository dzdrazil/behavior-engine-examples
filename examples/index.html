It's likely that you'll see a flash of content as the JavaScript is loading. This is due to the dynamic loading nature of the development-mode application, and should disappear (at least almost) entirely when running the minified code.

If that isn't sufficient, CSS will do the trick.

Reveal-If Behavior

This behavior should be used on an input, preferably a select type. It also should have the data-reveal-if attribute to specify which value the targets should toggle on.
Note that the data-reveal-if-target and data-reveal-if-not attributes on the target elements must match the name of the input; this enables the behavior to be reused multiple times on the same page.
